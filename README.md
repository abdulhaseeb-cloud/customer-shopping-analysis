# Customer Shopping Behavior Analysis

End-to-end Data Analytics project analyzing retail customer shopping trends.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL
- Power BI

## Key Insights
- Clothing dominates revenue at $104,264
- Fall is the strongest sales season
- 18-25 age group has highest average spend
- Montana is the top revenue location

## Files
- notebooks/eda.ipynb - Data cleaning and EDA
- sql/business_queries.sql - Business analysis queries
- customer_behavior_dashboard.pbix - Power BI dashboard
